# ⚔️ RPG Engine - JavaScript & Orientação a Objetos

Este repositório contém a resolução da Atividade Prática focada em **Programação Estruturada e Orientação a Objetos com JavaScript**. 

O projeto foi desenvolvido de forma incremental, modelando um sistema de combate de RPG em turnos para demonstrar a evolução dos conceitos de programação — desde funções básicas e variáveis até a aplicação de herança, polimorfismo e proteção de dados.

## 📂 Estrutura do Projeto

O projeto está dividido em quatro módulos principais, seguindo os requisitos da disciplina. Um arquivo central (`main.js`) orquestra a execução de todos os módulos.

```text
/ 
├── main.js
├── programacao_estruturada/
│   └── combateBasico.js
├── orientacao_objetos/
│   └── Personagem.js
├── visibilidade_encapsulamento/
│   └── PersonagemSeguro.js
└── encapsulamento_condicionais_heranca/
    └── ClassesEspecializadas.js
